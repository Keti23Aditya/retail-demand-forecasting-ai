# 🚀 AI Retail Analytics & Demand Forecasting

## 📌 Overview

This project is an end-to-end **Retail Sales Analytics and Demand Forecasting system** built using Python. It analyzes historical sales data, extracts meaningful insights, and predicts future demand using time series forecasting.

This type of solution is highly relevant for FMCG companies like Nestlé, where accurate demand prediction helps in optimizing inventory, reducing wastage, and improving supply chain efficiency.

---

## 🧠 Key Features

* 📊 Data Generation and Handling
* 🧹 Data Preprocessing and Cleaning
* 📈 Exploratory Data Analysis (EDA)
* 🔮 Demand Forecasting using ARIMA Model
* 📉 Visualization of Sales Trends and Predictions
* 🗄️ SQL Queries for Business Insights

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Statsmodels, Scikit-learn
* **Database Queries:** SQL
* **Tools:** VS Code, GitHub

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/your-username/retail-demand-forecasting-ai.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Navigate to src folder:

```
cd src
```

4. Run the project:

```
python data_generator.py
python main.py
```
